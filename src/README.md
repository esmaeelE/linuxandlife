# لینوکس و زندگی

> کتابی برای کشف دنیای پشت لینوکس و درک بهتر از زندگی

این کتاب سعی می‌کند به خواننده ایده‌هایی در مورد زندگی و لینوکس بدهد.
چرا زندگی؟ چون لینوکس یک فلسفه است و برآمده از یک جامعه، و کسی که
می‌خواهد در آن موفق باشد باید درک فلسفی سالمی از زندگی داشته باشد.
